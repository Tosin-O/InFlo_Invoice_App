# 📱 InFlo Invoice App (Frontend)

A fully functional **mobile invoice application frontend** built with **React Native + Expo**. This project focuses on delivering a clean, intuitive, and responsive UI for creating and managing invoices without a connected backend.

---

## ✨ Features

* 🧾 Create and view invoices (UI only)
* 📊 Dashboard-style overview screens
* 💼 Clean business-oriented interface
* 📱 Fully responsive mobile design (Android & iOS)
* 🧭 Smooth navigation between screens
* 💡 Reusable components for scalability

---

## 🛠️ Tech Stack

* React Native
* Expo
* JavaScript / TypeScript (if applicable)
* React Navigation

---

## ⚠️ Note

This project is **frontend-only** and does not include backend integration or persistent data storage. All data is currently handled on the client side for UI demonstration purposes.

---

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/inflo-invoice-app.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the Expo development server:

   ```bash
   npx expo start
   ```

4. Run on your device using the Expo Go app or emulator.
